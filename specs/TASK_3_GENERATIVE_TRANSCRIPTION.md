# Задача 3: Генеративна Транскрипція

## Опис

Реалізувати систему генерації транскрипції "на льоту" для мовних пар, де одна з мов не є рідною для користувача (наприклад, українська для носія нідерландської).

## Вимоги

### 1. Проблема

Користувач (наприклад, носій нідерландської), який вивчає українську, може не знати кирилиці. Йому потрібна транскрипція українських слів нідерландськими символами (звуками).

### 2. Рішення: Матриця Звуків (Custom Rules)

- Створити кастомну систему правил відображення звуків однієї мови символами іншої.
- **Виключення**: Англійська мова (використовує свою стандартну транскрипцію, існуючий механізм).
- **Підхід**: Матриця правил (Mapping) для конкретних пар мов.
  - На першому етапі реалізувати базу для конвертації `UK (Source) -> Latin (Target: NL, DE, etc.)`.

### 3. Технічна Реалізація

- Створити сервіс `src/lib/services/transcriptionService.ts`.
- Функція `generateTranscription(text, sourceLang, targetLang)`.
- Якщо пара мов має визначені правила -> генеруємо транслетирацію.
- Інтегрувати у `gameState.svelte.ts` при створенні карток:
  - Якщо `showTranscription` увімкнено, і мова не EN -> пробуємо згенерувати транскрипцію через сервіс.

### Приклад (Концептуальний)

Слово: "Природній" (UK)
Цільова мова: NL (Dutch)
Логіка: `п`->`p`, `р`->`r`, `и`->`y/i`, ...
Результат: `pryrodniy` (адаптовано під читання нідерландцем).
