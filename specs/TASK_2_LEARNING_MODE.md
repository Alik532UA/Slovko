# Задача 2: Режим "Навчання" (Learning Mode)

## Опис
Додати новий режим гри для початківців, який автоматично показує підказки у безкінечному циклі.

## Вимоги

### 1. Кнопка "Навчання"
- **Розміщення**: У компоненті `GameStats.svelte`, біля кнопки звичайної підказки (`hint-btn`).
- **Іконка**: `School` або `GraduationCap` (або інша відповідна з `lucide-svelte`).
- **Поведінка**: Перемикач (Toggle). Вмикає або вимикає режим навчання.

### 2. Логіка Режиму
При активному режимі навчання:
1.  Автоматично спрацьовує підказка для випадкової пари слів.
2.  **Анімація**: Підсвічування пари відбувається повільніше (в 5 разів повільніше за звичайну підказку).
3.  **Циклічність**: Після завершення анімації однієї пари, автоматично запускається підказка для наступної (через коротку паузу або одразу).
4.  Процес триває нескінченно, доки користувач не вимкне режим.

### Технічні деталі
- Файл: `src/lib/components/game/GameStats.svelte` (UI).
- Файл: `src/lib/store/gameState.svelte.ts` (Логіка).
- Додати стан `isLearningMode` у `gameState`.
- Модифікувати CSS анімацію для класу підказки, додавши варіант `.hint-slow` (duration: 5s).
